Master multi-line flex layout Master multi-line flex layouts. Mastering a multi-line flex layout allows, for example, to arrange product images and product titles in multiple lines. It is assumed that you have read the Introduction to Flex Layout article.

Multi-line flex layout The flex layout can be multi-line. To make the flex layout multiple lines, first specify "flex-wrap: wrap" for the element that specified "display: flex".
