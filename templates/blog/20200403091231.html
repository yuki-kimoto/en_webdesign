    [CSS3] opacity --Set opacity --Introduction to CSS [CSS3 compatible] To set the opacity of an element, set opacity in the style sheet (CSS). The maximum opacity is 1 and the minimum opacity is 0. In other words, the closer it gets to 0, the more transparent it becomes. 
    / * Opacity 1-Same as default * /
opacity: 1;

/ * Opacity 0.8-Slightly more transparent and shiny * /
opacity: 0.8;

/ * Opacity 0.4-Increases transparency * /
opacity: 0.4;

/ * Opacity 0.1-fading * /
opacity: 0.1;

/ * Opacity 0-Completely transparent * /
opacity: 0;


Let's actually set the opacity. Those with 0 opacity disappear completely and are invisible.
