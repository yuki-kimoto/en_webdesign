Descendant Selector-Introduction to CSS [CSS3 compatible]

Descendant selector is a descendant relationship ID selector, class selector and element selector. In the following example, ".food ul" is specified to specify "ul" which is a descendant of ".food". The specification ".food li" specifies "li" which is a descendant of ".food".
