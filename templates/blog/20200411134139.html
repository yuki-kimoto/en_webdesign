    * --Universal selector --Introduction to CSS [CSS3 compatible] The universal selector represented by "*" is a selector that applies to all tags. This value is set as the default for all tags such as div, p, and a tags. 
    * {{
  margin: 0;
  padding: 0;
}


The universal selector is useful for  setting stylesheet (CSS) defaults .
